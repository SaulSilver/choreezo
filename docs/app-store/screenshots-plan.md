# iOS Screenshot Plan

Capture screenshots from a release-like build with realistic data.

## Capture Setup

1. Use a clean test apartment with at least 3 members.
2. Seed one full week of chores with mixed assigned and unassigned items.
3. Enable notifications for one user to show settings state.
4. Avoid debug labels, developer toggles, and placeholder names.

## Required Device Families

Upload the required iPhone screenshot sets shown in App Store Connect for your selected device support.

## Recommended Shot Sequence (5 Screens)

1. Profile setup with Sign in with Apple option visible.
2. Apartment create/join screen showing invite-code flow.
3. Weekly schedule day view with clear assigned and unassigned chores.
4. My Chores view focused on current user tasks.
5. Settings screen showing reminders and apartment context.

## Optional Screens (If You Upload More)

1. Edit assignment flow with member picker.
2. Expanded members list in settings.
3. Empty state when no chores are claimed.

## Quality Checklist

1. Use consistent locale (en-US) and consistent clock time.
2. Keep text readable; avoid crowding with long names.
3. Show meaningful, realistic examples of chores.
4. Ensure each screenshot highlights a different value point.
5. Verify no personal data appears in captures.
